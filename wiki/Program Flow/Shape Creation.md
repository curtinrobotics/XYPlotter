Shape creation gets [[SVG File]] data from [[Input Output]] and processes the elements into shape objects, extracting the elements attributes.
![[shapeCreation.svg]]
# Find Elements
Extracts a list of elements from the SVG file

# Extract Element Details
Extracts all the attributes in each element

# Create Shape List
Creates a list of [[Shape]] objects from the element list

# Create Shape
Creates [[Shape]] and adds attribut to it

# Output Create Shape Error
Creates log messages on the shape creation status
